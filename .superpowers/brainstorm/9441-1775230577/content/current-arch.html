analyzePages() — HTML files only analyzeContent() — regex scan analyzeTech() — package.json analyzeMedia() — file extensions

❌ no refinement — must get it right first try
📦 Content

Phase 3 — AI Call #5 (sequential)
🚀 Launch synthesizes digest from 1–4 → readiness score, gaps, launch items

Core problem
Each agent gets one shot with a static context window. Agent 1 must guess which files matter without being able to explore. If it guesses wrong, all downstream agents cascade-fail. The model can't ask "wait, what does this router.js actually do?" — it either has it or doesn't.
